# SDT621-Formative-Assessment-1

## 📂 Solution Structure

This repository contains a single Visual Studio solution with multiple independent projects, each corresponding to a specific question in SDT621 -- Formative Assessment 1.
Each project is a standalone application (Console or Windows Forms) and should be run independently.

## ▶️ How to Run a Specific Project

All projects are part of **one solution**, but **only one project runs at a time**.

### Using Visual Studio

1. Clone this repository
2. In **Solution Explorer**, locate the project you want to run
3. **Right-click the project**
4. Select **Set as Startup Project**
5. Press **Run** (or `Ctrl + F5`)

Repeat these steps for each question/project you would like to test.
